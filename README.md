### 📊 [Ola Ride Bookings Analysis](https://github.com/Zishan9309/Ola_Ride_Bookings_Analysis)
> **SQL · Power BI · DAX · Data Analytics**

Analyzed 103K+ Ola ride bookings (Bangalore, July 2024) to uncover patterns in booking success, cancellations, revenue, and customer/driver satisfaction.

- 62% booking success rate identified; top cancellation reasons mapped by driver & customer
- ₹3.5 Cr+ revenue tracked across 7 vehicle types with Power BI interactive dashboard
- 10 business problems solved using MySQL Views
